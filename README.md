# Oracle Pluggable Database (PDB) Administration

## Assignment Overview

This assignment demonstrates Oracle Multitenant Architecture using Oracle Database 21c. The tasks include creating and managing Pluggable Databases (PDBs), creating database users, configuring Oracle Enterprise Manager (OEM), and documenting the practical work using GitHub.

---

## Oracle Environment

- Oracle Version: Oracle Database 21c Express Edition
- Operating System: Windows
- Tools Used:
  - Oracle SQL Developer
  - SQL*Plus
  - Oracle Enterprise Manager (OEM)
  - GitHub
